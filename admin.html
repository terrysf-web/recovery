Doctor-friendly editor for post-op recovery instructions.
Editing, deleting, uploading, and saving are locked until the admin password is entered.
Password recovery:
If the admin password is forgotten, open Cloudflare Worker settings and update the Secret named ADMIN_PASSWORD. That secret is the master recovery password. After using it to login, you can set a new admin password from this page.

Change Admin Password

Use the current admin password or the Cloudflare master recovery password to change it.
If the password is forgotten, reset the Cloudflare Worker Secret named ADMIN_PASSWORD. That secret works as the master recovery password.
