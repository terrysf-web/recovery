Staff Mode Generator

This adds a small "Staff Code Generator" button to the patient access screen. Staff can generate patient codes directly from a phone without opening the full admin dashboard.

1. Add Staff Button

<button class="staffModeBtn" onclick="openStaffMode()">
  Staff Code Generator
</button>

2. Add CSS

.staffModeBtn{
  width:100%;
  margin-top:12px;
  border:0;
  border-radius:16px;
  padding:13px;
  font-size:14px;
  font-weight:900;
  background:#e2e8f0;
  color:#334155;
}

3. Add JavaScript

function openStaffMode(){
  location.href='./code.html';
}

Result

Patient screen:

- Enter code
- Open Recovery Guide
- Staff Code Generator

Staff taps the button and opens the mobile generator page.
